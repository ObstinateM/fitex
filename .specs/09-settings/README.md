# Settings

## Purpose

Central configuration page for all user preferences. Also serves as a navigation hub to Stories and Import Stories pages.

## Access

- Gear icon (top-right corner), visible on Selector, Results, and History pages
- Not visible on: Onboarding, Settings itself, Stories, Import Stories

## Behavior

- Full-page modal with sticky header (back arrow + "Settings" title)
- All changes require explicit "Save Settings" click
- Shows "Saved!" confirmation for 2 seconds after saving
- Remembers which page opened it (selector/results/history) for the back button
