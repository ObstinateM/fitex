# Modified LaTeX Source

## Overview

Expandable section showing the raw LaTeX source of the tailored CV. Useful for debugging or manual editing.

## UI

- Gray-themed toggle button: "Modified LaTeX Source"
- Triangle expand indicator
- Only shown when `result.modifiedTex` exists

### Expanded Content
- `<pre>` block with monospace formatting
- Max height: 256px with scroll overflow
- Gray background tint
- Full raw LaTeX source displayed
